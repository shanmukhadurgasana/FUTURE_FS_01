# Shanmukha's Portfolio

A personal portfolio website for **Sana Shanmukha Durga**, built with vanilla HTML, CSS, and JavaScript using [Vite](https://vitejs.dev/) as the build tool.

## Features

- Responsive design for all screen sizes
- Dark / Light mode toggle with preference saved to `localStorage`
- Typing animation cycling through role titles (Tech Enthusiast, Web Developer, Full Stack Developer)
- Mobile-friendly navigation with hamburger menu
- Sections: Home, About, Skills, Projects, Resume, Contact
- Google Fonts (Poppins) and Font Awesome icons

## Project Structure

```
FUTURE_FS_01/
├── index.html    # Main HTML file with all page sections
├── main.js       # JavaScript (typing animation, theme toggle, mobile menu)
├── style.css     # Stylesheet
└── package.json  # Project config and scripts
```

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or later recommended)

### Installation

```bash
npm install
```

### Development

Start the local dev server with hot reload:

```bash
npm run dev
```

### Build

Generate a production build in the `dist/` folder:

```bash
npm run build
```

### Preview Production Build

Serve the production build locally:

```bash
npm run preview
```
